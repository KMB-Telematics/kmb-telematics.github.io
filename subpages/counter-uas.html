Counter Drone / Counter UAS (C-UAS)

Radar is a vital part of an integrated counter-drone / counter-UAS (cUAS) system. Radar is the only sensor that can see autonomous GPS-guided / non-RF emitting drones in all weather, day or night. KMB's unique imaging radar approach is able to detect and classify targets that other radars cannot. KMB radar provides rich data that can be processed with machine learning (ML) / AI to classify targets, reject false positives, or integrate with other sensors such as cameras / EO/IR.

KMB software-defined imaging radar is a generation ahead of the current generation of drone detection radar. A generation ahead of phased arrays Software-defined, flexible architecture We can see stationary, hovering drones We can track more than 20 targets at a time We can scan the airspace faster than 1 Hz We can distinguish birds from drones We are able to see drones near clutter: for example near trees or near the ground We are able to see drones in urban environments, where there may be very bright manmade radar reflectors
